## Requirements

- Python >= 3.13

## Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd billextractor
   ```

2. Install dependencies:
   ```bash
   # If using uv (recommended)
   uv sync

   # Or using pip
   pip install .
   ```

## Usage

Run the main script:

```bash
python main.py
